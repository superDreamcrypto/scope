About US

What Group Scope is currently capable of:

  • New Users can create an account
  • Existing user can sign in
  • Users can create and join groups
  • Users can edit their profile information
  • Users can visually see where group members are via google maps
  • Users can add multiple devices to their account
  • Admins can perform CRUD operations via the admin panel
